# Word Tracker

A simple tool to help you track your writing progress.

## Features

- Track your writing progress
- Export your writing as Markdown
- View your writing statistics
  - Words
  - Characters
  - Sentences
  - Paragraphs
  - Reading Time
  - Speaking Time
  - Reading Level
  - Frequency of Phrases
- Autosave

## Tech Stack

- React
- TypeScript
- Mantine
- Tiptap
